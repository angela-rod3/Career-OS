# Career-OS

CareerOS is a full-stack career management platform designed to help students organize and optimize their internship and job search process. Instead of tracking applications across spreadsheets, notes apps, emails, and reminders, CareerOS provides one centralized system for managing: 
* Internship and job applications
* Recruiter and referral relationships
* Resume versions
* Interview preperation
* Follow-up reminders
* Career analytics and progress tracking

# Tech Stack Overview

Frontend
* Angular
* TypeScript

Backend
* Python
* Django
* Django REST Framework (DRF)

Database
* PostgreSQL

Authentication
* JWT Authentication

Deployment
* Frontend: Vercel / Netlify
* Backend: Render / Railway / AWS
* Database: PostgreSQL Hosting
